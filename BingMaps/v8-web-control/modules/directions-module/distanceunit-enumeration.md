# DistanceUnit Enumeration
 The `Microsoft.Maps.Directions.DistanceUnit` enumeration has the following options.

| Name         | Description               |
|--------------|---------------------------|
| `km`         | A distance in Kilometers  |  
| `miles`      | A distance in Miles       |
